- **State Estimation** — Unscented Kalman Filter (UKF) providing sub-meter tracking accuracy for tumbling debris
- **Robotic Intervention** — 4-DOF carbon-fiber manipulator using Inverse Kinematics + Jacobian-based singularity avoidance for physical "Dock-and-Dump" debris capture
- **Physics-Correct Digital Twin** — Fully modeled in NASA GMAT with J2–J8 gravitational perturbations and real-time 209m² solar array power cycles
- **Control Law** — Model Predictive Control (MPC) for fuel-optimal trajectory planning under orbital constraints
